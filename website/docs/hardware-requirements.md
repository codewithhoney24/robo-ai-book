# Hardware Requirements

To successfully complete this course, you will need specific hardware for both simulation and potential physical robot implementation.

## Recommended System Specifications

### For Simulation Work:
- **CPU**: Intel i7 or AMD Ryzen 7 (8 cores or more)
- **RAM**: 16GB or more
- **GPU**: NVIDIA RTX Series (RTX 3060 or higher recommended)
- **Storage**: 500GB SSD
- **OS**: Ubuntu 22.04 LTS or Windows 10/11

### For Physical Robot Projects:
- **Robot Platform**: Humanoid Robot Kit (e.g., NAO, OP3, or custom)
- **Computer Vision Setup**: RGB-D Camera (Intel RealSense or equivalent)
- **Sensors**: IMU, LIDAR (optional but recommended)
- **Control Hardware**: Microcontroller board (Arduino/Raspberry Pi)

## Software Requirements
- ROS 2 Humble Hawksbill
- NVIDIA Isaac Sim (requires compatible RTX GPU)
- Unity 3D (Personal/Hobbyist license)
- Python 3.8+
- Docker (for reproducible environments)

## Specialized Equipment for Advanced Modules
- **Module 3**: Access to NVIDIA RTX-enabled workstation for Isaac Sim
- **Module 4**: Manipulator arm or multi-degree-of-freedom robot
- **Capstone**: Full humanoid robot platform for final implementation

## Alternative Setup for Cloud-Based Development
- Cloud GPU instances with RTX capabilities
- Remote development environment access
- Network connectivity for robot teleoperation
