# Assessment and Capstone Project

## Course Assessment Structure

### Continuous Assessment (60%)
- Weekly assignments and exercises: 20%
- Module projects and implementations: 25%
- Midterm presentation and review: 15%

### Final Capstone Project (40%)
- Autonomous humanoid robot demonstration: 30%
- Technical documentation and reflection: 10%

## Module-wise Breakdown

### Module 0: Introduction to Physical AI
- Quiz on Physical AI fundamentals: 5%
- Sensor systems hands-on assignment: 5%

### Module 1: The Robotic Nervous System (ROS 2)
- ROS 2 node programming exercise: 10%
- Multi-agent coordination project: 10%

### Module 2: Digital Twin & Simulation
- Gazebo simulation environment creation: 10%
- Unity sensor simulation implementation: 10%

### Module 3: The AI-Robot Brain (NVIDIA Isaac)
- Isaac Sim synthetic dataset creation: 8%
- ROS-VSLAM navigation system: 7%
- Reinforcement learning project: 10%

### Module 4: Vision-Language-Action (VLA)
- Humanoid kinematics implementation: 8%
- Manipulation and HRI design: 7%
- Conversational robotics system: 10%

## Capstone: Autonomous Humanoid Robot (40%)

### Objectives
Students will integrate all modules to develop a functional autonomous humanoid robot capable of:
- Navigation in a known environment
- Human-robot interaction
- Task execution based on verbal commands
- Obstacle avoidance and path planning

### Deliverables
1. **Technical Implementation**: Working humanoid robot demonstrating all required capabilities
2. **Documentation**: Comprehensive technical documentation including design decisions, challenges faced, and solutions implemented
3. **Video Presentation**: Demonstration video showing all required capabilities
4. **Reflection Paper**: Critical analysis of the sim-to-real transfer challenges encountered

### Evaluation Criteria
- **Functionality**: Does the robot successfully demonstrate all required capabilities?
- **Engineering Quality**: Implementation quality, code organization, and documentation
- **Innovation**: Creative solutions and novel approaches
- **Problem Solving**: Ability to overcome technical challenges
- **Integration**: How well different modules are integrated together
